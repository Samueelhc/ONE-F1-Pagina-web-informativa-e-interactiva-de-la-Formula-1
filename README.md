# 🏎️ F1 Web - Plataforma Informativa e interactiva de Fórmula 1

F1 Web es una página web desarrollada como proyecto académico en conjunto a otros compañeros, que presenta información relacionada con el mundo de la Fórmula 1. 

El sitio integra distintas secciones organizadas para ofrecer contenido estructurado sobre la temporada, equipos, pilotos, noticias y estadísticas, aplicando principios de diseño web y desarrollo frontend.

---

## 🚀 Funcionalidades

- Noticias actualizadas en tiempo real mediante la API de Motorsports.  
- Información detallada de escuderías y pilotos, incluyendo enlaces a tiendas oficiales y páginas de los equipos.  
- Sección interactiva de los monoplazas de cada escudería para consultar datos generales de los autos.  
- Calendario de la temporada 2025 con ubicación de cada carrera en el mapa mundi.  
- Resultados de todas las carreras desde la temporada 2020 a 2025 usando la API de Jolpica.  
- Histórico de carreras memorables de la F1: más victorias, más participaciones y otros récords destacados.  
- Chatbot integrado mediante la API de Gemini, dedicado exclusivamente a responder preguntas sobre Fórmula 1; rechaza temas ajenos.

---

## 🛠️ Tecnologías Utilizadas

- HTML
- CSS
- JavaScript
- Diseño responsive
- Estructuración modular del contenido

---

## 🎯 Objetivo

Desarrollar una plataforma web informativa aplicando buenas prácticas de diseño, organización de contenido y desarrollo frontend, fortaleciendo habilidades en maquetación, estilos y lógica en JavaScript.
